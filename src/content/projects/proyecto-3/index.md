## Descripción

Intervención urbana para revitalizar un parque público mediante arquitectura efímera y sostenible.

## Concepto Arquitectónico

Estructura modular de bambú que proporciona sombra y espacio para eventos comunitarios. Diseñado para ser desmontable y reutilizable en diferentes ubicaciones.
